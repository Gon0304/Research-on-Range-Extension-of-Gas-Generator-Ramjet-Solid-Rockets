## Candidate Baseline Missile Models

본 연구에서는 공개 제원 기반의 전술 탄도미사일을 baseline model로 사용한다.  
실제 무기체계의 상세 성능 재현이 아니라, 공개된 길이, 직경, 질량, 추진 방식, 사거리 정보를 기반으로 한 개념적 궤적 해석 모델을 구성한다.

### Primary Candidates

| Model | Reason |
|---|---|
| ATACMS-class model | Original baseline model with compact solid rocket configuration |
| LORA-class model | Similar mass and diameter to ATACMS, suitable for comparison |
| Hyunmoo-2B-class model | Larger SRBM-class model suitable for extended propulsion study |

### Secondary Candidates

| Model | Reason |
|---|---|
| Tochka-class model | Simple SRBM model for boost-coast trajectory validation |
| Iskander-class model | High-performance comparison case |
| Lance-class model | Historical comparison model before ATACMS |
| Scud-class model | Large SRBM reference case |
